# AGENTS.md

## Project

NBA Stats Explorer

A web application that allows users to search and explore NBA players, teams, game results, and statistics from the entire history of the NBA using the balldontlie API.

---

# Product Vision

Provide a fast, searchable NBA statistics database covering:

- Every NBA player available through the API
- Historical and current players
- Season averages
- Individual game logs
- Historical game results
- Regular season and playoff games
- Data dating back to the earliest seasons supported by the API
