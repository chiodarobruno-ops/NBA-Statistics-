# NBA Stats Explorer

NBA Stats Explorer is a historical NBA statistics application powered by the balldontlie API.

Users can search NBA players, browse season averages, view game logs, and explore historical game results across NBA history.

---

## Features

### Players

✅ Search any NBA player

✅ Current and historical players

✅ Player profiles

✅ Paginated search results

---

### Season Statistics

✅ Season averages

✅ Any available season

✅ Career exploration

Displayed metrics:

- Points
- Rebounds
- Assists
- Steals
- Blocks
- Turnovers
- FG%
- 3PT%
- FT%
- Minutes
- Games Played

---

### Game Logs

✅ Individual game statistics

Displayed metrics:

- Points
- Rebounds
- Assists
- Steals
- Blocks
- Turnovers
- FG%
- 3PT%
- FT%
- Minutes
- Plus / Minus

---

### Historical Games

✅ Historical game results

✅ Season filtering

✅ Team filtering

✅ Playoff filtering

Displayed information:

- Date
- Matchup
- Final Score
- Season
- Game Type
